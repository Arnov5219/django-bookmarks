# Django Bookmarks

A Django web application that allows users to bookmark images from any website using a JavaScript bookmarklet. Users can register, manage their profiles, and authenticate using Google OAuth.

## Features

- User registration and authentication
- Login and logout
- Password reset
- User profile management
- Google OAuth authentication
- Bookmark images from external websites
- JavaScript bookmarklet
- Image storage and management
- Responsive user interface

## Tech Stack

- Python 3
- Django 5
- HTML5
- CSS3
- JavaScript
- SQLite
- Google OAuth
- social-auth-app-django
